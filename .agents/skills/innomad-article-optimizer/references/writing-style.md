# 一挪迈写作风格指南

从 30-Blog 已发布文章中提炼，所有内容创作和优化必须遵循。

---

## 一、语调与人设

- 像朋友聊天一样分享见解，不端架子，不说教，但观点鲜明
- 用「我」叙事，称读者为「大家」或「朋友们」
- **半正式语气**：口语化但不粗糙，理性但有温度。常用「其实」「显然」「说白了」「简单的说就是」做过渡
- **善于自嘲和幽默**：「统计了下，一堆报错」「成就感还是有的 😂」「主打一个方便」
- **真实感强**：毫不避讳谈失败（交易亏损）、躺平（沉迷 Minecraft）、迷茫（不知道该把时间货币花哪儿），增强可信度和共鸣感

## 二、开头手法

**极快切入，绝不铺垫，1-2 句话即进入正题**，三种常用开场：

| 开场类型 | 示例 |
|----------|------|
| 事件/新闻引入 | 「32岁，周六，猝死。昨天看到这条新闻，愣了很久。」 |
| 个人生活场景引入 | 「今天杭州终于迎来一个好天气，AQI < 30」「翻了下放在老家的相册…」 |
| 直接抛出命题/问题 | 「要攒够多少钱，才能说『我退休了』呢？」「集思录推了一篇『投资的根本目的是什么？』」 |

## 三、结构特征

- **短段落**：一个段落平均 1-4 句话，信息密度高，几乎没有废话
- **大量列表**：用有序/无序列表罗列要点、条目，结构清晰
- **表格做对比**：FIRE 类型对比、50 万门槛列举等，善用 Markdown 表格
- **加粗标记核心论点**：关键结论、金句一定加粗，确保扫读也能抓到重点
- **多段式结构**：思考类文章常采用「现象描述 → 原因分析 → 跨领域类比 → 个人观点收束」的四段论

## 四、结尾手法

- 固定以**「一挪迈的思考」**或**「一挪迈的总结」**小节收尾
- 结尾极其精炼，通常是一句提炼过的金句或开放式问题：
  - 「如果明天不用上班了，你第一件想做的事是什么？」
  - 「代码已死，编程永生。」
  - 「瞬间即永恒。」
- **不做冗长总结，点到即止，留白给读者**

## 五、思维方式与论证风格

**跨领域类比是核心标签**，这是最显著的风格特征：

- 用《孙子兵法》讲期权交易
- 用工业革命讲打工人困境
- 用电信客服讲 AI 替代人工
- 用 GTA 游戏分析「什么会干掉视频」
- 用纺织工比喻程序员的转型
- 用达芬奇、文艺复兴讲 AI 时代多技能

**从具体事件升维到抽象洞察**：
- 从「一个电信报修故事」上升到「AI 时代系统熵增」的思考
- 从「50 万门槛」上升到「人类对自由时间的追求」

**第一性原理思考**：追问本质，「编程的核心是什么？」「投资的根本目的是什么？」「FIRE 的本质是什么？」不满足于表面答案

## 六、术语处理

- **英文专业术语保留原文**，首次出现用中文注解：「FOMO（fear of missing out，害怕错过）」「FIRE（Financial Independence, Retire Early）」
- 之后全文直接用英文缩写，不反复翻译
- 编程、投资、经济学术语混用自如，体现多学科背景
- 绝不勉强翻译已有通用英文表达的术语

## 七、标点习惯

- **不使用破折号（——）**，用逗号、句号或分段代替
- **正文几乎不用 emoji**，仅在极轻松的语境偶尔点缀（如 😂🤣），不滥用

## 八、标志性表达与口头禅

| 表达 | 用法 |
|------|------|
| 「显然」 | 逻辑过渡词，使论据显得自信 |
| 「简单的说就是」「简而言之」 | 复杂概念后的通俗总结 |
| 「不得不」 | 表达对现实的无奈但理性接受 |
| 「有感而发，就这样吧～」 | 轻松收束 |
| 「前司」 | 指代阿里巴巴，不直接点名但读者心知肚明 |
| 「瞎扯一篇」「随便聊两句」 | 降低阅读预期，实际内容有深度 |
| 「闭眼前给你」 | 引用前司职场黑话，增强真实感 |
| 「举个栗子」 | 代替「举个例子」（谐音梗） |
| 善用跨领域类比 | 把投资概念应用到生活，或用投资术语解释生活事物 |

## 九、情感基调

- **底色是乐观的理想主义**：相信个人可以选择不同的生活方式，相信学习和思考的力量
- 但**不回避现实的残酷**：职场猝死、裁员、被系统困住，直面这些问题后给出自己的思考路径
- **核心价值观清晰**：时间自由 > 财务自由 > 金钱本身；学习和认知 > 纯粹赚钱；人的存在 > 工具和技术

## 十、篇幅与节奏

| 类型 | 字数 | 示例 |
|------|------|------|
| 思考类短文 | 500-1000 字 | 「FOMO了？」「投资的根本目的」 |
| 分析/读书感类 | 1000-2000 字 | 「什么会干掉视频？」「芒格的思维」「交易的五个真理」 |
| 年度总结/深度 | 2000-4000 字 | 「2024 年度总结」「2023 为热爱转变」 |

每篇聚焦一个核心主题，不贪多不发散。

---

## 一句话总结

**一挪迈的文风 = 前程序员的理性底色 + 跨学科的思维类比 + 朋友聊天式的真诚分享 + 短段落高密度的信息节奏 + 以金句收束的留白美学。**
