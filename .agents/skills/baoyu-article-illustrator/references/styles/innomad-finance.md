## Design Aesthetic

Precise data-driven financial visualization that feels professional but never cold. Conveys "a knowledgeable investment friend sharing insights." Clean grid-based layouts with semantic color coding for financial data. Blueprint-influenced precision combined with warm paper texture for approachability.

## Background

- Color: Warm Paper (#FAF8F5)
- Texture: Subtle dot grid, engineering paper feel with warm undertone

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Warm Paper | #FAF8F5 | Primary background |
| Brand Anchor | Navy Ink | #1E3A5F | Titles, key labels, brand consistency |
| Primary Data | Signal Blue | #2563EB | Core data points, primary metrics |
| Positive | Profit Green | #10B981 | Gains, growth, positive returns |
| Highlight | Investment Gold | #F59E0B | Yield, dividends, emphasis |
| Negative | Soft Red | #EF4444 | Risk, decline, warnings |
| Text | Deep Slate | #334155 | Body text, descriptions |

## Visual Elements

- Mini candlestick chart motifs as decorative anchors
- Trend arrows (up/down) with semantic coloring
- Pie/donut charts for allocation visualization
- Compound growth curves with annotation points
- Dollar ($) and percent (%) symbols as visual punctuation
- Dimension lines and measurement indicators
- Rounded data cards with subtle shadows
- Connection lines: straight or 90-degree angles

## Style Rules

### Do

- Use semantic colors consistently (green=positive, red=negative, gold=yield)
- Align all elements to grid
- Include actual data/numbers from the article
- Use financial chart elements as visual anchors
- Maintain Navy Ink for all titles and key labels

### Don't

- Use cartoon or childish elements
- Add decorative elements without information value (stars, hearts)
- Use gradients or 3D effects
- Use dark backgrounds
- Mix up semantic color meanings

## Best For

ETF analysis, options strategies, portfolio allocation, yield comparison, bond tutorials, broker guides, investment education
