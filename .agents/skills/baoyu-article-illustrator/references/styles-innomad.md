All three styles share **Navy Ink `#1E3A5F`** as brand anchor color for cross-domain visual consistency.

## Style Characteristics by Type

### infographic + innomad-finance
- Data-driven layout with semantic color coding (green=gains, red=risk, gold=yield)
- Mini chart motifs (candlesticks, pie charts) as visual anchors
- Grid-aligned data cards with actual numbers from article
- Navy Ink titles, warm paper background, professional yet approachable

### comparison + innomad-finance
- Split layout with clear financial metric separation
- Color-coded sides using semantic finance colors
- Data tables and percentage comparisons prominent
- Dimension lines connecting compared items

### flowchart + innomad-finance
- Step-by-step process with financial context
- Each step in rounded data card with semantic border color
- Straight or 90-degree connection arrows
- Dollar/percent symbols as visual punctuation

### infographic + innomad-tech
- Modular rounded rectangles representing system components
- Directional arrows showing data flow between modules
- Terminal window and code bracket decorative elements
- Indigo/Cyan color scheme on cream background

### framework + innomad-tech
- Node-graph architecture with clear hierarchy
- Connection lines showing dependencies and data paths
- Gear/API icons at integration points
- Neural network cluster motifs for AI concepts

### flowchart + innomad-tech
- Linear or branching process with module nodes
- Each step as rounded rectangle with uniform outline
- Cyan arrows for data flow direction
- Terminal/code elements integrated into step descriptions

### scene + innomad-life
- Warm travel journal composition with lifestyle icons
- Globe, laptop, coffee cup as recurring motifs
- Sunset warm palette with gentle gradients
- Hand-drawn quality, personal and inviting atmosphere

### timeline + innomad-life
- Organic progression with dashed travel route lines
- Location markers at each timeline point
- Warm color temperature throughout
- Palm tree, compass, airplane route decorative elements
